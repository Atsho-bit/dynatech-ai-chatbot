# DynaTech AI Chatbot – Educational Conversational Assistant

**Live Demo:** https://cdn.botpress.cloud/webchat/v3.6/shareable.html?configUrl=https://files.bpcontent.cloud/2026/04/23/11/20260423111610-ZUSWV6RD.json  
 

## Overview

DynaTech is an educational AI chatbot that teaches fundamental AI concepts and assesses user understanding through multiple-choice quizzes. Built on Botpress Cloud and deployed via a shareable webchat link.

---

## Key Features

Users prompt an AI topic they want to learn. The bot delivers a concise lesson, then offers a topic-based multiple-choice quiz. Upon completion, users receive a closing acknowledgement message.

Topics covered include Machine Learning, Deep Learning, NLP, Computer Vision, Neural Networks, AI Ethics, Prompt Engineering, and Generative AI.

---

## Platform & Implementation

The chatbot is built on Botpress Cloud, chosen for its visual flow builder, native quick-reply buttons, conversation state management, and fully managed hosting with no infrastructure required. The free tier is used, and deployment is instant via a shareable webchat link.

Implementation followed a three-step approach: topic menu design with quick-reply buttons, dedicated lesson and quiz flows for each topic, and a quiz completion message ending the conversation.

---

## Conversation Flow

The user opens the chatbot and receives a welcome message. The user prompts the bot for topics. The user selects a topic, and the bot delivers a short educational lesson. The bot then presents multiple-choice quiz questions one by one. After the final question, the bot displays: "Well done, you have completed the quiz."

This linear structure is identical across all topics; only the content changes.

## Sample Q&A Pairs

**Machine Learning:** What is Machine Learning? Answer: A type of AI where systems learn from data. Which is an example in everyday life? Answer: Email spam filters. Which is a type of ML learning from labeled data? Answer: Supervised Learning.

**Deep Learning:** What is Deep Learning? Answer: A type of AI using layered neural networks to learn from data. Key building block? Answer: Neural Networks. Real-world application? Answer: Image recognition in smartphones.

**Natural Language Processing (NLP):** What does NLP stand for? Answer: Natural Language Processing. Common use? Answer: Translating languages automatically. Key challenge? Answer: Understanding sarcasm and jokes.

**Computer Vision:** What is Computer Vision? Answer: Teaching computers to understand and interpret images or videos. Real-world use? Answer: Facial recognition in smartphones.
